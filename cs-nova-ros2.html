Nova: ROS 2 Jazzy Differential Drive Robot

A custom-built, simulated differential-drive mobile robot designed as a scalable foundation for modern ROS 2 robotics applications, integrating precision kinematics with advanced physical simulation.

See Visual Proof View Repository
Project Insight
Seamless ROS 2 & Gazebo Harmonic Integration

Objective

Design Challenge

Developing a reliable simulation environment for mobile robots requires seamless integration between kinematic models and modern physics engines, moving away from deprecated legacy plugins to ensure future-proof scalability.

Goal

  • Develop a parametric "Digital Twin" utilizing URDF/Xacro for precise mechanical and kinematic representation.
  • Establish robust, bi-directional communication between ROS 2 Jazzy and Gazebo Harmonic.
  • Create a modular architecture primed for future implementations of SLAM and autonomous navigation (Nav2).

Process & Tech Stack

01 Architecture & Modeling

Engineered a professional ROS 2 package hierarchy. The mechanical core (nova_description) handles URDF, meshes, and RViz configurations, while the dynamic layer (nova_gazebo) manages world environments and spawning nodes.

02 Simulation Bridge

Implemented the modern ros_gz_bridge for flawless bi-directional data transport, complete with a carefully configured coordinate transform (TF2) tree (odom -> base_footprint -> base_link -> wheels).

Tools & Software

Simulation Gazebo Harmonic
Visualization RViz2

The Nova project successfully establishes a modern, highly scalable robotic simulation framework. By leveraging the latest ROS 2 Jazzy stack and Gazebo Harmonic, the system delivers an accurate physical representation and real-time kinematic monitoring. This robust, modular foundation bridges mechanical design with intelligent software, ensuring the differential-drive robot is perfectly positioned for the upcoming roadmap phases of LIDAR/camera perception, occupancy-grid mapping (SLAM), and autonomous waypoint navigation.
